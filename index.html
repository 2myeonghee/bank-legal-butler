01

기획 목적 및 추진 배경

1.1 아파트집사 플랫폼의 현재 구조

아파트집사는 공동주택 입주예정자협의회(입예협)를 대상으로 질의 등록 → 인덱스 매칭 → 변호사 검토 → 답변 게시 → 레퍼런스 축적의 흐름으로 운영되는 AI+변호사 하이브리드 법률자문 플랫폼이다. 집단등기·하자·건축 법률 등 공동주택 도메인에 특화된 RAG 인덱스와 반복 질문 자동 처리 구조가 핵심 자산이다.

핵심 발견
플랫폼의 기술 구조(질의 구조화 → 인덱스 라우팅 → 변호사 할당 → SLA 관리 → 레퍼런스 축적)는 도메인에 무관하게 이식 가능하다. 현재 등기팀이 저축은행 등에 제공하는 수작업 자문을 이 구조로 대체하면 변호사 공수는 줄이면서 단가는 유지할 수 있다.

1.2 B2B 금융권으로 확장하는 이유

공급 측 근거 — 로펌 입장
  • 현재 등기팀 → 저축은행 자문이 수작업·구두 형태로 레퍼런스 미축적, 담당자 교체 시 노하우 소멸
  • 변호사 1인이 반복 질문에 매번 신규 리서치 → 시간당 원가 대비 수익성 낮음
  • 정액 구독 수익이 없어 매출 변동성 높음 → 월정액 구조로 안정 수익 확보 가능
  • 아파트집사 인프라 개발비 이미 투입 → 추가 개발 최소화로 ROI 극대화
수요 측 근거 — 금융기관 입장
  • 새마을금고 3,242개, 저축은행 79개사 — 대부분 전담 법무팀 없음
  • 법률 이슈 발생 시 단건 의뢰 → 비용 예측 불가, 대응 지연
  • 반복 질문(등기·연체·담보 처리)에 매번 착수금 지불 → 비효율
  • 2021년 금소법 시행 이후 상시 법률 대응 필요성 급증
  • 지점 담당자 수준에서 처리 가능한 법률 판단 기준 부재 → 본점 에스컬레이션 과다

03

금융권 주요 법률 질문 유형 분석

아래 질문들은 황해 등기팀이 저축은행·새마을금고를 대상으로 실제 수행한 자문 경험과 금융권 실무 레퍼런스를 바탕으로 도출했다. 분류 기준은 빈도(월 발생 횟수) · 반복성(유사 질문 재발 가능성) · 난이도(변호사 개입 필요 여부)이다.

A. 여신 실행·계약 해석

대출 계약 체결 전후 발생하는 질의. 표준 약관 해석은 인덱스화 가능하나, 개별 계약 조항 분쟁·차주 특수 상황은 변호사 개입이 필요하다.

#실제 질의 내용빈도난이도반복성등급
A-01기한이익 상실 사유 발생 후 채무자에게 즉시 통보하지 않으면 이의 제기할 수 있나? 여신거래기본약관상 통보 의무 확인매우 높음낮음높음L1
A-02근저당권 설정 시 채권최고액 적정 비율 — 업권별 권장 기준(저축은행 130%, 새마을금고 120% 등)매우 높음낮음매우 높음L1
A-03법인 대출 시 대표이사 연대보증 요구 가능 여부 — 상법 542조의9(상장사 임원 보증 금지)·금소법 적용 범위 판단높음중간높음L2
A-04포괄근담보 설정 — 2020년 이후 개인 포괄근보증 금지 규정이 담보물 설정에도 영향을 미치나?높음중간높음L2
A-05사업자 명의 대출인데 실질이 가계자금인 경우 — DSR 규제 회피로 간주될 리스크 및 취급 가이드중간높음중간L2
A-06차주 신용등급 하락 시 추가 담보 청구 요건 — 여신거래기본약관 제5조 해석, 차주가 거부 시 기한이익 상실 발동 가능한가?높음중간높음L2
A-07만기 연장 심사 중 연체 발생 — 갱신 거절 통보 서면 요건 및 차주 이의신청 대응중간중간높음L2
A-08차주 사망 후 상속인이 3개월 내 상속 포기한 경우 — 채권 처리 전략 및 상속재산관리인 선임 필요 여부중간높음중간L2
A-09변동금리 대출에서 기준금리 상승 시 차주에게 사전 통지 의무 범위 — 금소법 정보 제공 의무와의 관계높음중간높음L2
A-10이사회 결의 없이 집행된 새마을금고 고액 여신 — 대외 여신 효력에 영향을 미치나? 이사 개인 책임 범위 (새마을금고법 제25조, 제29조)낮음매우 높음낮음L3

B. 담보 권리 실행·경매·명도

담보 부동산 경매 개시부터 낙찰·명도까지 전 과정. 임차인 배당 예측, 낙찰 후 점유자 처리, 유찰 반복 대응이 가장 빈번한 실무 이슈다.

#실제 질의 내용빈도난이도반복성등급
B-01임의경매 신청 전 채무자에게 별도 최고(催告) 해야 하나? — 상환 촉구 없이 곧바로 경매 신청 가능 여부매우 높음낮음매우 높음L1
B-02임차인 있는 담보물 경매 — 최우선변제 소액보증금 기준(2023년 개정) 및 배당 예측, 차순위 채권 회수 가능 금액 시뮬레이션매우 높음중간높음L2
B-03경매 낙찰 후 점유자 강제 퇴거 — 명도소송(6개월~1년+) vs 인도명령(2~3개월) 선택 기준과 비용 비교높음중간높음L2
B-04담보 부동산에 세입자의 대항력 유무 확인 — 전입신고일·확정일자와 근저당 설정일 비교, 대항력 있는 임차인 처리 전략매우 높음중간매우 높음L2
B-05선순위 가압류 이후 근저당 설정한 경우 — 경매 배당에서 가압류권자와 우선순위 충돌 해소 방법중간높음중간L2
B-06경매 최저가 3회 유찰 후 — 공매 전환(한국자산관리공사) vs 담보가등기 실행(청산 절차) 비교 및 선택 기준중간높음중간L2
B-07복수 담보(공동근저당) 설정 후 일부만 경매 신청 — 나머지 담보에 미치는 영향 및 이중배상 방지 방법중간높음중간L2
B-08경매 배당이의 — 선순위 임차인이 주장하는 보증금이 우리 파악 금액보다 클 때 이의신청 절차 및 증거 자료중간높음중간L2
B-09법인이 실질 폐업 상태인 담보물 — 임의경매 신청 가능 여부, 대표이사 개인 재산 병행 집행 방법중간높음낮음L2
B-10집합건물(상가) 담보 — 구분 소유권(호실)과 건물 전체 담보의 차이, 공용부분 경매 신청 가능 범위낮음매우 높음낮음L3

C. 채권 회수·소멸시효·NPL

소멸시효 실수 하나로 수억 원 채권이 소멸될 수 있다. 반복성이 높고 절차가 명확한 항목은 L1 자동화, 사해행위·파산 참여는 L3 처리.

#실제 질의 내용빈도난이도반복성등급
C-01소멸시효 기산점 — 마지막 이자 납입일부터 5년 경과 직전, 내용증명만으로 시효 중단 가능한가? (최고 효력 6개월 내 제소 의무)매우 높음낮음매우 높음L1
C-02이자 일부 납부(채무 승인)로 시효 중단 — 중단 효과 발생 시점과 새로운 기산점 계산높음낮음높음L1
C-03지급명령 신청 절차 — 채무자 이의신청 후 소송 전환 시 대응, 지급명령 vs 소 제기 직접 비용·기간 비교높음낮음높음L1
C-04급여 압류 및 전부명령 — 민사집행법 246조 압류 금지 금액 계산, 은행 예금 채권 압류+전부명령 절차높음낮음높음L1
C-05사해행위취소소송 — 채무자가 대출 직후 배우자에게 부동산 증여한 경우, 취소 가능 기간(5년) 요건 및 수익자 악의 입증 방법중간높음중간L2
C-06채무자 개인회생 신청 시 — 담보채권은 회생 계획안에서 어떻게 취급되나? 회생 중 경매 신청이 중지되나?중간높음중간L2
C-07채무자 파산 선고 후 배당 신고 — 배당신청 기한, 채권 증명 서류 목록, 담보채권자와 비담보채권자 배당 순서중간중간중간L2

7.2 실행 로드맵

Phase 1 · 0~3개월

파일럿 구축 및 첫 계약

기술
  • RAG 데이터 교체 (금융법 6종 파싱·임베딩)
  • 질문 입력 폼 재설계
  • 담당군 라우팅 4개 그룹 설정
  • L1 인덱스 30~50건 초기 구축
영업·운영
  • 기존 등기팀 거래 저축은행 1~2개사 파일럿 제안
  • 3개월 무료 파일럿 → 유료 전환 조건 명시
  • 모든 질의 L2 처리 → 레퍼런스 50건 이상 축적
  • 파일럿 완료 후 성과 자료화
Phase 2 · 3~9개월

L1 자동화 가동 및 고객 확장

기술
  • L1 자동화 가동 (목표: 전체 질의의 40%)
  • 화이트라벨 고객 포털 (기관별 로고·색상)
  • 월 정기 법률 리포트 자동 생성
  • 규제 변경 자동 알림 (금융위·금감원 공고 RSS)
영업·운영
  • 저축은행·새마을금고 5~10개사 Standard 계약 목표
  • 파일럿 고객 추천 프로그램 운영
  • 월 구독 수익 500만원 이상 달성
  • 2차 타깃(상가연합회) 파일럿 1개 착수
Phase 3 · 9~24개월

플랫폼 안정화 및 타깃 확장

기술
  • L1 자동화율 60% 이상 달성
  • Enterprise 전담 변호사 배정 시스템
  • 2차 도메인(상가·프랜차이즈) RAG 추가
  • API 연동 — 고객사 내부 시스템 연결
영업·운영
  • 고객 20개사 이상 / 월 수익 2,000만원 목표
  • 새마을금고 중앙회 연계 — 지점 일괄 계약 추진
  • 3차(중소기업) 시장 진입 검토
  • 타 로펌 라이선스 판매 검토

뱅크법률집사는 아파트집사의 기술 자산을 재활용하여 최소 개발 비용으로 금융권 B2B 구독 수익을 창출하는 구조다. 초기 3개월 파일럿에서 레퍼런스를 빠르게 축적하여 L1 자동화율을 높이는 것이 수익성의 핵심이다. 변호사 공수는 레퍼런스가 쌓일수록 자동으로 줄어드는 선순환 구조를 가지며, 1차 목표는 월 100만원 Standard 고객 10개사 달성 → 월 1,000만원 안정 구독 수익이다.
